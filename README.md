# Retoucher

Retoucher is a freelance marketplace platform for photographers, photo editors, retouchers, studios, and AI-based image services.

## Main Features

- User registration and authentication
- Photographer and customer orders
- Editor profiles and portfolios
- Bidding system
- Order management
- Secure payment workflow
- Wallet and commission system
- Admin supervision
- Multi-language support
- AI image services in future phases

## Tech Stack

- Backend: Django + Django REST Framework
- Frontend: Next.js + TypeScript
- Database: PostgreSQL
- Cache/Queue: Redis + Celery
- Storage: Local first, S3-compatible later
- Deployment: Docker, Nginx, Gunicorn
